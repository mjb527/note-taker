Enter a new note!

GET

Endpoint: /api/notes

This will return all notes, including:

  • Their ID
  • Their Name
  • Their time & date
  • Their description
  • Their location
Endpoint: /api/notes/:id
Returns all notes with the ID or Name mathcing the parameter `name` This will return data all matches, including:
  • ID
  • Name
  • Time & date
  • Description
  • Location

POST

Endpoint: /api/notes/
Create a new note by passing along the following parameters:
  • name (string): the required name displayed to you when you view the list of notes
  • date (string): the optional date for the note in case it is an event, formatted MM/DD/YYYY
  • time (string): the optional time for the note in case it is an event, formatted HH:MM AM/PM
  • description (string): the optional long description if it is too long to be appropriate for the title
  • location (string): the optional location of an event
Endpoint: /api/notes/delete/:id
Delete a single note by passing its ID. Returns an array of the remaining objects.
